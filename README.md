router 설정


(global router)

/ → 이것은 Homepage가 될 것이다.
/join → 회원가입할 때 Join이다.
/login → 로그인을 하기 위한 login 페이지이다.
/Search → 검색 페이지인 Search 이다.

(user router)

/users/edit → 유저가 프로필을 수정하는 페이지이다.
/users/delete → 유저가 프로필을 삭제하는 페이지이다.

(video router)

/videos/watch → 동영상 시청 페이지이다. 
/videos/edit → 동영상 수정 페이지이다. 
/videos/delete → 동영상 삭제 페이지이다.
/videos/comment → 동영상 댓글 페이지이다 
/videos/comments/delete → 동영상 댓글 삭제 페이지이다.
